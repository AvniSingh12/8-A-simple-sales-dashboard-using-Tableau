# 8: Basic Sales Dashboard Design

Task:
Develop a sales dashboard that is interactive using Tableau to compare Superstore sales performance based on product category, region, and month.

Dataset:
- File: `Sample - Superstore.csv`
- Columns used: Order Date, Region, Category, Sales, Profit

Features of Dashboard:
- Line graph illustrating monthly sales trends 2014 to 2017
- Bar graph of total sales by region
- Donut graph illustrating distribution of sales by category
- Interactive filter by Region

Key Insights:
1. The West region had the highest sales throughout.
2. Technology was the best-selling category in revenue.
3. Monthly sales were highest around November of each year.
4. The most profitable in terms of sales was Central region.

Files Included:
- `Superstore_Sales_Dashboard.twbx` – Tableau workbook
- `dashboard_superstore_sales.png` – Dashboard export
- `insights.txt` – Key business insights
- `README.md` – Description of the project

Tools Used:
- Tableau Public (No paid software used)

Learning Outcome:
- Created an end-to-end interactive dashboard
- Experienced data visualization and storytelling with Tableau
